Subjective Hallucination Scale (SHS) Calculator - Demo

Basic Usage

This is a simple example of the SHS calculator embedded in a page.

Usage Example

To use the calculator in your own page:

<!-- Include CSS --> <link rel="stylesheet" href="shs-calculator.css"> <!-- Include JavaScript --> <script src="shs-calculator.js"></script> <!-- Create container --> <div id="my-shs-calculator"></div> <!-- Initialize --> <script> const calculator = new SHSCalculator('my-shs-calculator', { language: 'en', // 'en', 'de', or 'fr' showGauge: true, showConsistency: true }); // Listen for results document.getElementById('my-shs-calculator') .addEventListener('shs:calculated', (e) => { console.log('SHS Results:', e.detail); }); </script>

API Methods

Available Methods:
  • calculator.setLanguage(lang) - Change language ('en', 'de', 'fr')
  • calculator.getResults() - Get current results object
  • calculator.reset() - Reset form and clear responses
// Get results programmatically const results = calculator.getResults(); console.log('Overall Score:', results.overall); console.log('Breakdown:', results.breakdown); // Change language calculator.setLanguage('de'); // Reset form calculator.reset();

Results Structure

The results object contains:

{ overall: -0.5, // Overall score (-1 to +1) overallConsistency: 0.2, // Overall consistency score breakdown: [ // Per-dimension breakdown { dimension: {...}, // Dimension pair info score: -0.5, // Dimension score consistency: 0.2, // Dimension consistency label: "Factual Accuracy" }, // ... more dimensions ], responses: { // Raw responses q1: 2, q2: -2, // ... all 10 questions } }
